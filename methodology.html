Methodology

This project explores language use and English proficiency across US states using publicly available census-based datasets.

Data Sources

We combine state-level language counts with 2010 population estimates to compute proportions and compare patterns across states.

Processing Steps

Speaker counts are normalized by state population to compute percentages. Visualizations aggregate counts by state and language.

Visualization Notes

Charts are built with D3.js and update interactively where applicable. All maps use US state boundaries from the US Atlas TopoJSON.
